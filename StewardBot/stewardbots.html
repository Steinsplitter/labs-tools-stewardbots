StewardBot - Help

StewardBot (aka Stewie) is nice little bot who spends his life in #wikimedia-stewards. He makes life easier for the stewards and everyone else; he is a good bot!

Commands

Command Description Notes/Example
!steward Pings all of the stewards (emergency)
@steward Pings a group of stewards who have opted-in (non-emergency) <kibble>@steward Az has gone rouge!
<StewardBot>Stewards: Attention requested by kibble ( DerHexer Dungodung )
@quiet Makes Stewie be quiet and not speak. :'( <dungodung>@quiet
<StewardBot>I'll be quiet :(
@speak Makes Stewie talk again. <dungodung>@speak
<StewardBot>Back in action :)
@mlock Make Stewie be quiet for 10 seconds. <dungodung>@mlock
<StewardBot>You have 10 seconds!
@notify on Turns notification (!steward) on <StewardBot>Notification on
@notify off Turns notification (!steward) off <StewardBot>Notification off
@stew
@privileged
@ignored
@stalked
@listen		 Advanced settings See the section below for more information.
@help Displays a link to this help page <kibble>@help
<StewardBot>Help = https://tools.wmflabs.org/stewardbots/stewardbot.html
@huggle The huggle command. :D
@test Tests the rc reader and posts what was last received on irc.wikimedia.org
@die Quits the bot (owner-only) <dungodung> @die
* StewardBot has left irc.freenode.net ("Goodbye!")

Advanced settings

Command Subcommand Parameters Description Notes/Example
stew add username [nick [cloak [opt-in]]] Adds a new steward defined by username and optional parameters: nickname, cloak and boolean opt-in information; special value "-" can be used for the cloak parameter if the user has no cloak <dungodung>@stew add Cbrown1023 kibble wikimedia/Cbrown1023 true
remove
delete		 username Removes a steward (and all related info) by username <dungodung>@stew remove Cbrown1023
change
edit
modify
rename		 username [username [nick [cloak [opt-in]]]] Changes the username, nick, cloak and/or opt-in information of a steward; special value "-" can be used for all but the first parameter <dungodung>@stew change Cbrown1023 - worby - 0
users Lists all stewards by username <dungodung>@stew users
nicks Lists all stewards by nicknames <dungodung>@stew nicks
optin Lists all stewards who have opted-in for @steward notification by nicknames <dungodung>@stew optin
info user Provides information about the specified steward <dungodung>@stew info Cbrown1023
<StewardBot>Steward Cbrown1023 uses nick kibble with the cloak wikimedia/Cbrown1023. kibble is in the list of opt-in nicks.
privileged add nick cloak Adds a nick to the list of privileged users <kibble>@privileged add Az1568 wikipedia/Az1568
remove
delete		 nick Removes a nick from the list of privileged users <kibble>@privileged remove Az1568
change
edit
modify
rename		 nick (nick|cloak) Changes the nick or the cloak of a privileged user <kibble>@privileged edit Az1568 wikimedia/Az1568
list [(all|*)] Lists all privileged users; with optional parameter, lists stewards as well <kibble>@privileged list *
get nick Cloaks who can give restricted commands to the bot <kibble>@privileged get Az1568
<StewardBot>The cloak of privileged user Az1568 is wikimedia/az1568
ignored add username Adds a user to the list of ignored users <kibble>@ignored add Az1568
remove
delete		 username Removes a user from the list of ignored users <kibble>@ignored remove Az1568
change
edit
modify
rename		 username username Changes the username of an ignored user <Az1568>@ignored modify kibble worby
list [(all|*)] Lists all ignored users; with optional parameter, lists stewards as well <kibble>@ignored list all
stalked add page Adds a page prefix to the list of stalked pages <kibble>@stalked add Steward_handbook
remove
delete		 page Removes a page prefix from the list of stalked pages <kibble>@stalked remove Steward_handbook
change
edit
modify
rename		 page page Changes the page name of a stalked page <kibble>@stalked rename Steward_handbook Stewards'_handbook
list Lists all stalked pages <kibble>@stalked list
listen add channel Adds a channel to the list of 'listen' channels <kibble>@listen add #cvn-sw
remove
delete		 channel Removes a channel from the list of 'listen' channels <kibble>@listen remove #cvn-sw
change
edit
modify
rename		 channel channel Changes the channel name of a 'listen' channel <kibble>@listen rename #cvn-sw #cvn-swmt
list Lists all 'listen' channels <kibble>@listen list
on Joins all 'listen' channels <kibble>@listen on
off Parts all 'listen' channels <kibble>@listen off

Note: username and page parameters should always use underscore (_) instead of space.
